LLM Wiki Starter — 완전 사용 설명서

자료를 넣을수록
더 똑똑해지는 AI 지식 뇌

AI가 내 맥락을 기억하고, 자료가 쌓일수록 더 정확하게 답하는 지식의 복리 시스템입니다. 개발자가 아닌 분도 사용할 수 있습니다.

📥 자료 누적🔗 개념 연결 🧠 AI 맥락 기억📈 지식 복리
무엇이 다른가요?
❌ 기존 방식의 문제
• AI에게 매번 맥락을 새로 설명
• 자료가 흩어져 연결 안 됨
• 좋은 답변도 다음 대화에서 사라짐
• 두 번째 질문이 첫 번째보다 나아지지 않음
✅ 이 시스템의 해결책
• wiki/에 지식이 누적되어 맥락이 영구 보존
• 개념 간 연결이 자동으로 형성됨
• 좋은 답변이 다음 질문의 출발점이 됨
• 자료가 쌓일수록 AI가 더 정확하게 답함

첫 번째 자료 넣기

이제 자료를 AI에게 넘겨 wiki로 만들 차례입니다. 두 가지 방법 중 하나를 선택하세요.

🌐 방법 A — claude.ai 웹 (가장 쉬움)
🖥️ 방법 B — Claude Code 앱 (추천)
이런 분께 적합: 일단 먼저 써보고 싶은 분, 설치 없이 시작하고 싶은 분
A1

claude.ai 접속 후 새 대화 시작

브라우저에서 claude.ai를 열고 로그인 → "새 대화" 클릭

A2

파일 첨부 후 요청

📎 아이콘을 눌러 CLAUDE.md와 내 자료 텍스트 파일을 첨부하고 아래처럼 입력합니다:

C
Claude
📄 CLAUDE.md
📄 내 자료.txt
CLAUDE.md 규칙을 읽고, 첨부한 자료를 wiki로 정리해줘. wiki/ 구조로 페이지를 만들고 ontology도 업데이트해줘.
메시지 입력...
A3

결과 저장

Claude가 생성한 wiki 페이지 내용을 복사해서 wiki/ 폴더 안에 .md 파일로 저장합니다.

⚠️ 웹 방식은 파일을 직접 저장해야 합니다. Claude Code 앱을 쓰면 자동 저장됩니다.
이런 분께 적합: 매일 쓰는 분, 자동화를 원하는 분, wiki 파일을 자동으로 저장해주길 원하는 분
B1

Claude Code 앱 다운로드 & 설치

claude.ai 웹사이트에서 Claude Code 데스크톱 앱을 다운로드합니다.

B2

앱 열고 프로젝트 폴더 지정

Claude Code 앱을 열고 wiki-llm-starter 폴더를 프로젝트로 지정합니다. CLAUDE.md를 자동으로 읽어 규칙을 인식합니다.

B3

raw/ 폴더에 자료 넣기

자료 텍스트 파일을 templates/llm-wiki/raw/ 폴더에 넣습니다.

📖 용어 사전
이 시스템에서 자주 나오는 단어들을 쉽게 설명합니다. 모르는 단어가 나오면 여기서 찾아보세요.
스킬
Skill
AI에게 "이런 방식으로 일해줘"라고 미리 정해둔 작업 지시서입니다. 레시피처럼 AI가 어떤 순서로 무엇을 해야 하는지 적혀있습니다.
예: "wiki-ingest 스킬"을 부르면 → 자료 읽기 → 페이지 생성 → 검증 → 저장을 순서대로 자동 실행합니다.
플러그인
Plugin
기본 기능에 추가로 붙이는 확장 도구입니다. 스마트폰 앱처럼, AI에게 없던 능력을 추가해줍니다.
예: gbrain 플러그인을 추가하면 AI가 wiki 페이지를 의미 기반으로 검색할 수 있게 됩니다.
MCP
Model Context Protocol
AI가 외부 도구(파일, 검색 등)를 사용할 수 있게 연결하는 통신 규격입니다. USB 포트처럼 표준화된 연결 방식입니다.
예: "gbrain MCP를 연결했다" = gbrain 검색 기능을 Claude가 쓸 수 있게 됐다.
온톨로지
Ontology
개념들의 분류 체계와 관계 규칙입니다. "이 세계에는 어떤 종류의 것들이 있고, 어떤 관계가 가능한가"를 정의합니다.
예: "Tool은 Workflow에서 사용된다" — 이런 규칙들이 모인 것이 온톨로지.
T-Box
Terminological Box
온톨로지의 설계도입니다. 어떤 클래스(종류)가 있고, 어떤 관계가 허용되는지 규칙을 정의합니다.
파일: ontology/tbox.json — 여기에 규칙이 저장됩니다.
A-Box
Assertional Box
온톨로지의 실제 데이터입니다. T-Box 규칙에 맞는 실제 사례(인스턴스)들이 저장됩니다.
파일: ontology/abox.json — AI가 자료에서 발견한 개념들이 여기 쌓입니다.
Vault
Vault
하나의 주제를 담는 폴더 묶음입니다. raw/, wiki/, ontology/를 포함한 전체 구조를 말합니다.
예: "my-ai-wiki"는 AI 관련 자료만 담는 vault.
Ingest
Ingest
원본 자료를 AI가 분석해서 wiki 지식 페이지로 변환하는 과정입니다. "소화"와 비슷한 개념입니다.
raw/에 파일 넣기 → wiki-ingest 실행 → wiki/에 정리된 페이지 생성.
raw/
raw directory
원본 자료 보관함입니다. AI가 절대 수정하지 않습니다. 유튜브 자막, 기사, 메모를 텍스트로 넣는 곳입니다.
권장 파일 형식: .txt, .md
wiki/
wiki directory
AI가 raw/ 자료를 분석해서 개념별로 정리한 지식 페이지가 저장되는 폴더입니다. Obsidian으로 열면 그래프로 볼 수 있습니다.
wiki/concepts/, wiki/sources/, wiki/questions/ 등 하위 폴더로 구분됩니다.
gbrain
gbrain
wiki 페이지를 의미 기반으로 검색하는 도구입니다. 정확한 단어가 없어도 관련 내용을 찾아줍니다. (선택 설치)
"컨텍스트 윈도우"를 검색하면 그 단어가 없어도 관련 페이지를 찾아줍니다.
Claude Code
Claude Code
컴퓨터에 설치하는 Claude 데스크톱 앱입니다. 파일을 직접 읽고 쓸 수 있어서 wiki 파일 자동 저장이 가능합니다.
웹 claude.ai보다 더 많은 기능을 자동으로 처리해줍니다.
Codex
Codex
OpenAI의 AI 코딩 도구입니다. 이 시스템의 스킬은 Codex에서도 동일하게 작동하도록 설계되어 있습니다.
Codex에서는 $wiki-ingest 처럼 $ 기호로 스킬을 호출합니다.
터미널 / 검은 화면
Terminal / Command Prompt
텍스트로 컴퓨터에 명령을 내리는 검은 창입니다. 무서워 보이지만 명령어를 복사해서 붙여넣기만 하면 됩니다.
Windows: Win키 → "터미널" 검색 → 열기. gbrain 설치 시만 필요합니다.
Lint
Lint / 건강검진
wiki의 품질 검사입니다. 끊어진 링크, 출처 없는 주장, 고아 페이지 등을 자동으로 찾아줍니다.
"wiki-lint 해줘" → AI가 문제점 목록을 리포트합니다.

🧩 핵심 개념
이 시스템을 이루는 네 개의 레이어와 그 역할을 이해합니다.
📥

raw/ — 원본 자료 창고

모든 원본이 들어오는 곳. AI가 읽기만 하고 절대 수정하지 않습니다. 유튜브 자막, PDF 발췌, 메모를 텍스트로 넣으면 됩니다.

📚

wiki/ — AI 정리 지식 페이지

AI가 raw/ 자료를 읽고 개념별로 정리한 Markdown 페이지. 출처, 연결 링크, 핵심 주장이 구조적으로 정리됩니다.

🗺️

ontology/ — 개념 지도

tbox.json: 어떤 종류의 개념이 있는지 규칙. abox.json: 실제 자료에서 발견된 사례들. 잘못된 관계는 오류로 잡힙니다.

🔍

gbrain — 의미 검색 엔진

wiki/ 페이지를 벡터 + 키워드 하이브리드 검색으로 인덱싱합니다. 정확한 단어가 없어도 관련 페이지를 찾아줍니다.

1단계 · 누적

  • 관심 분야 자료를 꾸준히 raw/에 넣기
  • wiki-ingest로 매번 정리하기
  • log.md에 미해결 질문 남기기
  • 좋은 답변을 questions/에 저장하기

2단계 · 연결

  • 반복 등장하는 개념 찾기
  • ontology에 새 관계 유형 추가
  • wiki-query로 개념 간 연결 확인
  • 모순 주장을 contradiction으로 표시

3단계 · 합성

  • "A와 B의 공통점은?" 질문하기
  • 여러 vault를 gbrain에 함께 인덱싱
  • 교차점에서 새 아이디어 포착
  • 합성 통찰을 새 위키 페이지로 저장

4단계 · 진화

  • 반복 패턴을 schema/에 원칙으로 저장
  • 검증된 프레임워크를 ontology에 추가
  • AI가 내 사고방식을 학습하기 시작
  • 나만의 지적 시스템 완성

🗺️ AI가 나의 맥락 지도이자 나침반이 된다는 것

wiki/는 "AI에게 나를 설명하는 문서"입니다. 내가 어떤 개념을 중요하게 생각하는지, 어떤 자료를 신뢰하는지, 어떤 문제를 해결하려 하는지가 온톨로지와 위키 페이지에 담깁니다.

AI는 매번 이 지도를 보고 내 맥락에 맞는 답을 찾습니다. "지금 이 결정에서 내 원칙에 맞는 방향은?"을 물으면, 내가 과거에 저장한 결정 사항과 가치관을 바탕으로 나에게 맞는 방향을 제시합니다.
