맥락 영구 보존

AI에게 매번 새로 설명할 필요 없습니다. wiki/에 누적된 맥락을 자동으로 활용합니다.

🔗

개념 자동 연결

자료가 쌓일수록 개념 간 연결이 형성됩니다. 두 번째 질문이 첫 번째보다 더 나아집니다.

🏗️

온톨로지 레이어

T-Box/A-Box 분리로 개념 체계를 검증합니다. 잘못된 관계는 스크립트가 자동으로 잡아냅니다.

🔍

의미 기반 검색

gbrain 하이브리드 검색(Vector+BM25)으로 정확한 단어 없이도 관련 내용을 찾습니다.

⚡ 빠른 시작 (Claude Code)
5단계로 완전 설치 — 터미널에 붙여넣기만 하세요
# 1. gbrain 설치 (한 번만) bun install -g github:garrytan/gbrain gbrain init --pglite # 2. Claude Code에 연결 claude mcp add gbrain -- gbrain serve # 3. wiki 인덱싱 gbrain import templates/llm-wiki/wiki/
# 4. Claude Code에서 첫 ingest wiki-ingest로 raw 안의 자료를 넣어줘
✅ 터미널이 낯선 분은 → 완전 사용 설명서의 🚀 첫 걸음 탭에서 ZIP 해제부터 첫 사용까지 시각적으로 안내합니다.
