# Wiki Rules

## Page Types

- Source page: one raw source, its metadata, summary, and extracted claims.
- Concept page: a reusable idea that appears across sources.
- Entity page: a person, organization, product, or tool.
- Workflow page: a repeatable process.
- Question page: a useful answer that should compound back into the wiki.

## Required Sections

Every wiki page should include:

1. One-line definition
2. Summary
3. Source links
4. Related pages
5. Open questions

## Link Rules

- Use Obsidian-style links: `[[concepts/llm-wiki]]`.
- Link source pages from concept pages.
- Link concept pages from source pages.
- Do not create duplicate pages for the same concept unless the meaning is truly different.

## Claim Rules

- Every strong claim needs a source link.
- Mark uncertain claims as `uncertain`.
- Mark contradictions as `contradiction-candidate` until reviewed.
